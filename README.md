# Tools to Generate CTD Files for Archival in Findable and Searchable Repositories

## Background Motivation

Conductivity-Temperature-Depth (CTD) profilers are common tools in oceanography used in a wide variety of applications. While data is regularly collected by oceanographers, it doesn't always make it to searchable databases where it can be found and used by other research teams outside of those involved in the data collection. This limits scientific progress and monitoring, especially in rapidly changing areas. One roadblock between data collection and delivery to a searchable archive is the processing of data and metadata into a standard forward. In this repository, we provide a set of examples to transform different types of CTD data into organized NetCDF data with associated metadata that meets CF compliant standard.

## Who are these examples for?

These examples are for individual researchers or small oceanographic research groups who may lack dedicated specialists that format and deliver data to searchable archives.

## Understanding these examples

This repository contains examples for the converstion of three types of CTD files: Seabird `.cnv` files, RBR `.nc` files, and Tsk `.edf` files. There are equivalent scripts for use in both Matlab and Python.
